# Voice Form Filler

Fill form fields on any website using your voice.

## Build

```bash
npm install
npm run build
```

This creates the `dist/` folder.

## Load in Chrome

1. Open `chrome://extensions`
2. Turn on **Developer mode** (top right)
3. Click **Load unpacked**
4. Select the `dist/` folder

## Use

1. Click a form field on any website
2. Click the extension icon → **Start Recording**
3. Speak, then click **Stop Recording** — the text is inserted into the field
